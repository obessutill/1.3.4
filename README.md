Позитивные тесты:
1. Название: Ввод стандартного валидного email-адреса
   Шаги воспроизведения: 
   1) Ввести в поле ввода email "test@example.com"
   2) Нажать кнопку "Проверить"

13. Название: Отсутствие валидации при вводе текста в поле
    Шаги воспроизведения: 
    1) Ввести в поле ввода email "invalid email"
    2) Не нажимая кнопку «Проверить», продолжить ввод
    Ожидаемый результат: Сообщение об ошибке не отображается, alert не появляется

14. Название: Поведение формы при пустом поле ввода
    Шаги воспроизведения: 
    1) Оставить поле ввода email пустым
    2) Нажать кнопку "Проверить"
    Ожидаемый результат: Над кнопкой отображается ошибка красного цвета с текстом «Введен некорректный email»

15. Название: Исчезновение сообщения об ошибке при последующем вводе
    Шаги воспроизведения: 
    1) Ввести невалидный email "invalid"
    2) Нажать кнопку "Проверить"
    3) После появления ошибки начать вводить валидный email
    Ожидаемый результат: Сообщение об ошибке исчезает при начале ввода нового текста
